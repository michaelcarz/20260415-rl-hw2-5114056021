## 📋 作業目的

實作並比較兩種經典強化學習演算法 — **Q-learning**（Off-policy）與 **SARSA**（On-policy），透過相同的 Cliff Walking 環境與參數設定，分析其學習行為、收斂特性以及最終策略差異。

## 📊 結果分析

### 1. 學習表現

網頁提供**每回合累積獎勵曲線**，包含原始數據和滑動平均線，可以觀察到：
- SARSA 收斂更快且更穩定
- Q-learning 波動較大，偶爾出現大量負獎勵（因墜崖）

### 2. 策略行為

| 演算法 | 策略特性 | 路徑描述 |
|--------|----------|----------|
| Q-learning | 冒險策略 | 沿懸崖邊緣走最短路徑 |
| SARSA | 安全策略 | 遠離懸崖，繞行上方 |

### 3. 穩定性分析

透過**滑動標準差圖**可以看到：
- Q-learning 的標準差始終高於 SARSA
- SARSA 在收斂後波動極小

### 4. 多次實驗統計

透過 30 次以上獨立實驗的**均值 ± 標準差帶**，驗證結論的統計穩定性。

## 🔍 理論比較

### Q-learning（離策略 Off-policy）
```
Q(s,a) ← Q(s,a) + α[r + γ · max_a' Q(s',a') − Q(s,a)]
```
- 更新使用下一狀態的**最佳可能行動**（即使未實際執行）
- 學到理論最優策略，但訓練不穩定

### SARSA（同策略 On-policy）
```
Q(s,a) ← Q(s,a) + α[r + γ · Q(s',a') − Q(s,a)]
```
- 更新使用**實際採取的行動**
- 將探索風險納入考量，學到較安全的策略

## 📌 結論

1. **收斂速度**：SARSA 收斂較快
2. **穩定性**：SARSA 更穩定（標準差更小）
3. **最優性**：Q-learning 學到理論最優（最短路徑），但實際執行時因 ε-greedy 探索頻繁墜崖
4. **應用建議**：
   - 犯錯代價高 → 選 **SARSA**
   - 需要最優策略且可承受訓練風險 → 選 **Q-learning**
